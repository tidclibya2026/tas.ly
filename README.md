# TAS.LY – منظومة الإيواء السياحي الليبية

مشروع وطني رقمي لإدارة وحصر وتصنيف ومتابعة منشآت الإيواء السياحي في ليبيا.

## الهدف من المشروع

بناء منظومة موحدة تشمل:

- السجل الوطني لمنشآت الإيواء السياحي.
- إدارة الفنادق والقرى والمنتجعات والشقق الفندقية.
- إدارة التراخيص وأذونات المزاولة.
- متابعة التصنيف والجودة.
- التقارير الإحصائية للإشغال والليالي السياحية.
- لوحة مؤشرات لدعم القرار.
- الربط الجغرافي GIS.

## الجهة المالكة

مركز المعلومات والتوثيق السياحي  
وزارة السياحة والصناعات التقليدية – دولة ليبيا

## حالة المشروع

المرحلة الأولى: تنظيم المشروع وتجهيز النموذج الأولي.
