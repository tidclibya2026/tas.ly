# قاموس بيانات منظومة الإيواء السياحي TAS.LY

## 1. الهدف من قاموس البيانات

يوضح هذا الملف الجداول الأساسية في قاعدة بيانات منظومة الإيواء السياحي، ومعنى كل جدول، وأهم الحقول الموجودة فيه، واستخدامها داخل النظام.

---

## 2. جدول facility_types

### الغرض
تعريف أنواع منشآت الإيواء السياحي.

### أمثلة
- فندق
- قرية سياحية
- منتجع
- شقق فندقية
- نزل
- مصيف موسمي

### أهم الحقول

| الحقل | المعنى |
|---|---|
| id | رقم نوع المنشأة |
| name_ar | اسم النوع باللغة العربية |
| code | كود مختصر لنوع المنشأة |

## 14. ملاحظات مهمة

- جدول facilities هو الجدول المركزي للنظام.
- كل منشأة يجب أن يكون لها facility_code لا يتكرر.
- جدول facility_capacity منفصل لأن الطاقة الاستيعابية قد تتغير عبر الزمن.
- جدول classifications منفصل لأن التصنيف قد يتغير.
- جدول licenses منفصل لأن الترخيص يتجدد سنوياً.
- جدول monthly_occupancy هو أساس التقارير الإحصائية.
- جدول audit_logs مهم للرقابة والشفافية.
